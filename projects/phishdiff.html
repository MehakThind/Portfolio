PhishDiff — AI-Based Phishing Detection Tool

Tools Used: Python, FastAPI, OSINT, Machine Learning

Outcome

Developed a phishing detection system that analyzed sender details, content, and URLs. Produced explainable outputs that helped users understand why an email was flagged, improving analyst confidence and reducing false positives.

What I Learned

  • Explainability is as important as accuracy in security tools.
  • OSINT enrichment adds valuable context but needs caching to avoid delays.
  • Clear documentation makes technical tools more accessible to non-technical users.
